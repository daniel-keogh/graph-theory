match.cli module¶

Command Line Interface for the program.

Options List:
-h, --help Shows a help message and then exits.
-r, --regex The regular expression to match.
-t, --text The string to match the regular expression against.
match.cli.parse() → argparse.Namespace[source]¶

Parses the user’s command line arguments using argparse and then returns back to the caller an argparse.Namespace object that contains the user’s input arguments.

Returns:An argparse.Namespace object from where the parsed argument values can be retrieved.
